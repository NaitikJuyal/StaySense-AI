# StaySense AI


## Intelligent Guest Review Sentiment & Theme Analysis Platform

StaySense AI is an AI-powered web application designed to help homestay owners and hospitality businesses analyze guest feedback efficiently. The platform automatically classifies reviews by sentiment, identifies key themes, and generates professional response suggestions using Google Gemini AI.


## Features

* Sentiment Analysis (Positive, Neutral, Negative)
* Theme Detection (Food, Host, Location, Cleanliness, Value, Experience)
* AI-Generated Response Suggestions
* Bulk Review Processing
* Analytics Dashboard
* Review History Management

## Installation

1. Clone the repository

git clone https://github.com/NaitikJuyal/StaySense-AI.git

2. Install dependencies

Frontend:
npm install

Backend:
npm install

3. Configure environment variables

Create a .env file:

GEMINI_API_KEY=api_key
MONGODB_URI=mongodb_connection_string

4. Start the application

Frontend:
npm run dev

Backend:
npm start
